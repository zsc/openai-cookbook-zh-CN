Claude 技能

欢迎来到 Anthropic Cookbook 的技能部分！本目录包含一系列指南，展示了 Claude 擅长的特定技能和能力。每份指南都深入探讨了一项特定技能，讨论了潜在的用例、用于优化结果的提示工程技术以及评估 Claude 性能的方法。

指南

  • 使用 Claude 进行分类：了解 Claude 如何革新分类任务，尤其是在具有复杂业务规则和有限训练数据的场景中。本指南将引导您完成数据准备、使用检索增强生成 (RAG) 进行提示工程、测试和评估。

  • 使用 Claude 进行检索增强生成：了解如何使用特定领域的知识增强 Claude 的能力。本指南演示了如何从头开始构建 RAG 系统、优化其性能以及创建评估套件。您将了解摘要索引和重新排序等技术如何显著提高问答任务的准确性、召回率和整体精确度。

  • 使用上下文嵌入进行检索增强生成：了解如何使用一种新技术来提高 RAG 系统的性能。在传统的 RAG 中，文档通常被分成更小的块以进行有效检索。虽然这种方法适用于许多应用程序，但当单个块缺乏足够上下文时，可能会导致问题。上下文嵌入通过在嵌入之前向每个块添加相关上下文来解决此问题。您将学习如何将上下文嵌入与语义搜索、BM25 搜索和重新排序结合使用以提高性能。

  • 使用 Claude 进行摘要：探索 Claude 从多个来源摘要和综合信息的能力。本指南涵盖了各种摘要技术，包括多轮、基于域和分块方法，以及处理长格式内容和多个文档的策略。我们还探讨了评估摘要，这可能是一种艺术、主观性和正确方法的平衡！

  • 使用 Claude 进行文本到 SQL 转换：本指南介绍了如何使用提示技术、自我改进和 RAG 从自然语言生成复杂的 SQL 查询。我们还将探讨如何评估和提高生成 SQL 查询的准确性，并进行测试语法、数据正确性、行数等的评估。

开始使用

要开始使用技能指南，只需导航到所需指南的目录，然后按照 guide.ipynb 文件中提供的说明进行操作。每个指南都是独立的，并包含重现示例和实验所需的所有必要代码、数据和评估脚本。

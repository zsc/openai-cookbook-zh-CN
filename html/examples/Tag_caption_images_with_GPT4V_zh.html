标记图片

在本节中，我们将使用 GPT-4o mini 为我们的产品生成相关标签。

我们将使用一种简单的零样本方法来提取关键字，并通过嵌入来去重，以避免出现多个过于相似的关键字。

我们将结合使用图片和产品标题，以避免提取图片中描绘的其他物品的关键字——有时场景中会描绘多个物品，而我们只想标记我们想要的那个物品。

提取关键字

system_prompt = '''
    你是一个专门为家具、装饰品或陈设品进行标记的代理，用于在市场上搜索相关关键字。

    你将获得一张图片和图片中物品的标题，你的目标是仅提取指定物品的关键字。 

    关键字应简洁且小写。 

    关键字可以描述诸如：

    - 物品类型，例如“沙发床”、“椅子”、“书桌”、“植物”
    - 物品材质，例如“木材”、“金属”、“织物”
    - 物品风格，例如“斯堪的纳维亚”、“复古”、“工业”
    - 物品颜色，例如“红色”、“蓝色”、“白色”

    只有当物品材质、风格或颜色明显且能使图片中的物品脱颖而出时，才推断这些关键字。

    以字符串数组的格式返回关键字，如下所示：
    ['desk', 'industrial', 'metal']

'''

def analyze_image(img_url, title):
    response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[
        {
            "role": "system",
            "content": system_prompt
        },
        {
            "role": "user",
            "content": [
                {
                    "type": "image_url",
                    "image_url": {
                        "url": img_url,
                    }
                },
            ],
        },
        {
            "role": "user",
            "content": title
        }
    ],
        max_tokens=300,
        top_p=0.1
    )

    return response.choices[0].message.content

# 搜索 N 个最相似的结果
def search_from_input_text(query, n = 2):
    embedded_value = get_embedding(query)
    df_search['similarity'] = df_search['embedding'].apply(lambda x: cosine_similarity(np.array(x).reshape(1,-1), np.array(embedded_value).reshape(1, -1)))
    most_similar = df_search.sort_values('similarity', ascending=False).iloc[:n]
    return most_similar

使用 Assistants API (GPT-4) 和 DALL·E-3 创建幻灯片

本笔记本演示了如何使用新的 Assistants API (GPT-4) 和 DALL·E-3 来制作信息丰富且视觉吸引人的幻灯片。
创建幻灯片是许多工作的关键方面，但可能非常繁琐且耗时。此外，从数据中提取见解并有效地在幻灯片上阐述它们可能具有挑战性。

本指南将演示如何利用新的 Assistants API 来简化您端到端的幻灯片创建过程，而无需您接触 Microsoft PowerPoint 或 Google Slides，从而为您节省宝贵的时间和精力！

0. 设置

from IPython.display import display, Image
from openai import OpenAI
import os
import pandas as pd
import json
import io
from PIL import Image
import requests

client = OpenAI(api_key=os.environ.get("OPENAI_API_KEY", "<your OpenAI API key if not set as env var>"))

#Lets import some helper functions for assistants from https://cookbook.openai.com/examples/assistants_api_overview_python
def show_json(obj):
    display(json.loads(obj.model_dump_json()))

def submit_message(assistant_id, thread, user_message,file_ids=None):
    params = {
        'thread_id': thread.id,
        'role': 'user',
        'content': user_message,
    }
    if file_ids:
        params['file_ids']=file_ids

    client.beta.threads.messages.create(
        **params
)
    return client.beta.threads.runs.create(
    thread_id=thread.id,
    assistant_id=assistant_id,
)

def get_response(thread):
    return client.beta.threads.messages.list(thread_id=thread.id)

1. 创建内容

在本指南中，我们将为我们的公司 NotReal Corporation 的季度财务审查创建一个简短的虚构演示文稿。我们希望强调一些影响公司盈利能力的关键趋势。
假设我们有一些财务数据可用。让我们加载数据并查看一下……

financial_data_path = 'data/NotRealCorp_financial_data.json'

首先，我们需要上传文件，以便我们的助手可以访问它。

file = client.files.create(
  file=open('data/NotRealCorp_financial_data.json',"rb"),
  purpose='assistants',
)

现在，我们准备创建我们的助手。我们可以指示我们的助手充当数据科学家，并处理我们给它的任何查询，运行必要的代码来输出正确的数据可视化。这里的 instructions 参数类似于 ChatCompletions 端点中的系统指令，可以帮助指导助手。我们还可以启用 Code Interpreter 工具，以便我们的助手能够编写代码。最后，我们可以指定我们想要使用的任何文件，在本例中只是我们上面创建的 financial_data 文件。

assistant = client.beta.assistants.create(
  instructions="You are a data scientist assistant. When given data and a query, write the proper code and create the proper visualization",
  model="gpt-4-1106-preview",
  tools=[{"type": "code_interpreter"}],
  file_ids=[file.id]
)

现在让我们创建一个线程，并作为我们的第一个请求，要求助手计算季度利润，然后按时间绘制各分销渠道的利润图。助手将自动计算每个季度的利润，并创建一个结合季度和年份的新列，而无需我们直接要求。我们还可以指定每条线的颜色。

thread = client.beta.threads.create(
  messages=[
    {
      "role": "user",
      "content": "Calculate profit (revenue minus cost) by quarter and year, and visualize as a line plot across the distribution channels, where the colors of the lines are green, light red, and light blue",
      "file_ids": [file.id]
    }
  ]
)

现在我们可以执行我们的线程运行

run = client.beta.threads.runs.create(
    thread_id=thread.id,
    assistant_id=assistant.id,
)

我们现在可以开始一个循环，该循环将检查图像是否已创建。注意：这可能需要几分钟时间

messages = client.beta.threads.messages.list(thread_id=thread.id)
